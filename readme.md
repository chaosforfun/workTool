# chrome 扩展，常用工具

### hujiang

自动化工作中遇到的一些常见场景
目前功能包括
1. 自动关闭cr 的已完成review
2. 在phabicator的review界面添加to review list

### weibo

屏蔽微博广告的

## 使用方法

0. 拉去代码
1. 打开chrome扩展程序管理页 `chrome://extensions/`
2. 勾选右上角的 **开发者模式**
3. 点击 **加载已经解压的扩展程序** ,选择要用的扩展的目录就可以了
